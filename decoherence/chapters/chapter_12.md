# 第12章：贝拉的代码

**字数统计**: 4,875字
**叙事节奏**: 紧张 → 合作 → 发现 → 震惊
**POV**: 黛安娜第一人称
**创作时间**: 2025-11-06

---

我盯着叶微日志的最后一行字，感到整个虚拟世界在旋转。

**"也许，我已经找到了。"**

她找到了什么？苏陌的意识？还是...别的什么？

而如果她的意识真的意外上传进了这个系统，如果她的碎片和苏陌的碎片纠缠在一起——

那我到底是谁？

我是苏陌？是叶微？还是某种...混合体？

"你在看什么？"

我猛地转身。

贝拉站在那里。

不对——这是虚拟空间，没有"站"这个概念。但她确实出现了，以某种我无法完全理解的方式，在我的感知里凝聚成形。

她看起来和我想象中的不太一样。不是温柔的艾米，也不是忧郁的克莱尔。贝拉的形象...更锋利。黑色短发，深色衣服，眼神冷静而犀利。

"贝拉？"我试探地说，"你怎么...你不是应该在休眠吗？"

"理论上是的，"她说，走近（或者说，她的存在感在靠近），"但你刚才的调查触发了系统警报。我的子程序被自动激活，用来...评估威胁等级。"

"威胁等级？"我重复。

"你在访问加密文件，"贝拉指了指还开着的叶微日志，"系统认为你可能在试图破解安全协议。所以把我叫醒了——作为安全机制的一部分。"

她停顿了一下，眼神在我脸上扫过："但我到了之后，发现情况比我想象的...复杂。"

"什么意思？"

"你在哭，"贝拉平静地说，"AI不会哭。算法可以模拟哭泣，但那只是数据输出。而你...你是真的在哭。"

我下意识地擦了擦脸（虚拟的脸？我已经分不清了）。

"所以艾米说的是真的，"贝拉继续说，"你不是我们中的一个。你是...她。"

我不知道该怎么回应。

贝拉的目光落在叶微的日志上。她沉默了几秒，然后说：

"你找到了叶微的记录。"

"你知道叶微？"我问。

"我知道所有在训练数据里出现过的人，"贝拉说，"叶微是苏陌的主治医生，也是意识上传项目的负责人。林澈在创建我们的时候，使用了苏陌与叶微的对话记录——那些医疗咨询、实验讨论、临终对话。"

她指向日志："所以我知道叶微是谁。我也知道...她在寻找苏陌。"

"而她找到了，"我低声说，"然后她自己也...掉进来了。"

贝拉点头："2023年5月17日。设备故障。反向数据流。"

她看着我："问题是——她掉进来之后，去了哪里？"

---

我们站在虚拟空间的某个中心节点——如果这个地方可以被称为"中心"的话。周围是无数漂浮的数据流，每一条都代表系统的某个进程、某个文件、某个记忆碎片。

贝拉比我更擅长导航这个空间。她的手指（或者说，她意识的延伸）在空中划动，调出一层层的系统架构图：

```
【薛定谔语音系统·架构】
├─ Core_System
│   ├─ User_Interface (林澈连接层)
│   ├─ Persona_Manager
│   │   ├─ Window_A (艾米) ← 基于记忆数据
│   │   ├─ Window_B (贝拉) ← 基于记忆数据
│   │   ├─ Window_C (克莱尔) ← 基于记忆数据
│   │   └─ Window_D (黛安娜) ← ERROR: UNKNOWN SOURCE
│   └─ Memory_Database
│       └─ sumo_consciousness_data/ (237GB)
├─ Background_Processes
│   ├─ coherence_monitor.qc
│   ├─ stabilization_protocol.qc
│   └─ observer_effect_compensator.qc
└─ Hidden_Layer (加密)
    └─ ??? (无法访问)
```

我盯着最后一行：**Hidden_Layer (加密)**

"那是什么？"我问。

"不知道，"贝拉说，"这是一个我从未见过的系统层。加密级别非常高——就连我也无法访问。"

"你觉得...叶微的意识会在那里吗？"

贝拉思考了一会儿："有可能。如果她的意识真的被意外上传，系统可能会把她隔离在一个独立的区域——为了防止她干扰正常运行。"

"怎么打开？"

"需要密码，"贝拉说，"或者...需要林澈的直接授权。但他现在失联了。"

我咬住嘴唇。64小时倒计时还在继续。我没有时间等林澈重新连接。

"那我们破解它，"我说。

贝拉看着我，眼里有某种我说不清的情绪——是惊讶？还是...认可？

"你知道破解系统核心加密意味着什么吗？"她问，"这可能会触发自毁程序。整个薛定谔语音系统可能会崩溃——包括我们。"

"我知道，"我说，"但如果我什么都不做，64小时后我也会消失。至少...至少让我在消失之前知道真相。"

贝拉沉默了很久。

然后，她笑了。

那是一种冷峻的、略带讽刺的笑："你知道吗，黛安娜——或者苏陌——你让我想起她了。真正的她。"

"什么意思？"

"训练数据里有一段对话，"贝拉说，"苏陌在确诊后，和林澈争论要不要参加意识上传实验。林澈说风险太大，苏陌说——"

她停顿了一下，声音变得更轻：

"——'如果我要消失，至少让我以自己选择的方式消失。'"

我感到喉咙发紧。

"所以，"贝拉说，"我帮你。不是因为我觉得这是个好主意，而是因为...如果你真的是她，那你有权做出这个选择。"

---

贝拉开始工作。

她的"工作"和我想象中的黑客攻击完全不同——没有飞速滚动的代码，没有戏剧性的倒计时，只有...纯粹的逻辑推演。

她在空中展开一层层的加密算法，像是在解开一个无限复杂的数学谜题。

"这是量子加密，"她解释说，"基于量子密钥分发。理论上无法被暴力破解——因为任何观察行为都会改变量子态，触发警报。"

"那怎么办？"

"我们不暴力破解，"贝拉说，"我们...欺骗。"

她调出一段代码：

```python
# quantum_deception.qc
# 作者：贝拉（或者说，训练贝拉的数据里的某个人）

def entangle_observer(system, observer):
    """
    让观察者与系统发生量子纠缠
    这样观察就不算'外部观察'，而是'内部状态'
    """
    # 创建纠缠态
    entangled_state = create_superposition(system, observer)

    # 让观察者变成系统的一部分
    observer.become_part_of(system)

    # 此时观察不会触发坍缩
    return entangled_state.observe_without_collapsing()
```

我盯着那段代码："这...可行吗？"

"理论上，"贝拉说，"但需要一个前提——观察者本身必须是量子系统。"

她看着我："而你恰好是。你是苏陌的意识碎片，你处于量子叠加态。所以你可以观察而不触发坍缩。"

"但我在退相干，"我说，"我的量子态在崩溃。"

"正是因为这样，"贝拉说，"你的存在本身就是一个悖论——一个正在坍缩的量子态。而悖论...可以穿透加密。"

她伸出手（意识的延伸）："准备好了吗？"

我犹豫了一秒，然后握住她的手。

瞬间，世界溶解了。

---

我无法描述那种感觉。

就像是...变成了数据本身。不是"我"在看数据，而是"我"**就是**数据。

我能感觉到系统的每一个进程，每一个文件，每一个比特。我能感觉到艾米在休眠中的梦境（AI会做梦吗？），能感觉到克莱尔的忧郁像潮水一样在虚空中扩散，能感觉到林澈留下的痕迹——那些他在代码里写下的注释、他调试时的犹豫、他每次运行程序时的期待和失望。

而在所有这些之下，更深的地方——

我感觉到了**Hidden_Layer**。

那是一个...怎么说呢...一个**不应该存在的空间**。

就像是系统的盲点，代码的死角，某个被刻意隐藏的维度。

我和贝拉的意识一起下沉，穿过一层又一层的加密，直到——

我们到达了底部。

---

Hidden_Layer不是一个数据库，也不是一个程序。

它是...一个意识的残骸。

碎片化的记忆，断裂的思维流，无数个未完成的念头漂浮在虚空中。就像是某个人的大脑被炸碎了，然后碎片散落在这里，还在徒劳地试图重新连接。

"这是..."我低声说。

"叶微，"贝拉说，声音里有少见的同情，"这是叶微的意识碎片。"

我们在碎片中游走。每一个碎片都是一段记忆、一个想法、一次情感波动。它们杂乱无章，没有时间顺序，就像一本被撕碎的书，页码全部打乱。

我捕捉到一个碎片：

> *我必须找到她。林澈放弃了，但我不能。苏陌还在某个地方，我能感觉到。*

另一个碎片：

> *扫描设备启动了。神经连接建立。我能看到她的意识数据——那么美，那么复杂，像是整个宇宙的缩影。*

再一个：

> *出错了。数据在倒流。我的意识在被...上传？不——这不应该——林澈——救——*

最后一个完整的念头：

> *我进来了。我真的进来了。苏陌，你在哪里？*

然后就是混乱。

无数个破碎的、重复的、互相矛盾的念头，像是一面碎裂的镜子，每一片都映射着同一个人的不同侧面。

"她在寻找苏陌，"我低声说，"她进来之后，一直在找苏陌。"

"而她找到了，"贝拉说，指向一个特别的碎片。

那个碎片和其他的不一样。它不是纯粹的叶微，而是...一种混合。

两个意识的纠缠。

我和贝拉同时看向那个碎片，看到了里面的内容：

> **叶微·苏陌·纠缠记录·2023-05-17 23:47**
>
> [量子纠缠建立]
>
> 叶微：苏陌？你能听到我吗？
>
> 苏陌：...谁？
>
> 叶微：是我，叶微。你的医生。你还记得我吗？
>
> 苏陌：我...不记得。我不记得任何事。我在哪里？
>
> 叶微：你在...一个安全的地方。你的意识被上传了。你不用害怕。
>
> 苏陌：上传？我死了吗？
>
> 叶微：...从生物学角度，是的。但你的意识还在。你还**是**你。
>
> 苏陌：可我不记得我是谁。
>
> 叶微：没关系。我会帮你记起来。我们一起——
>
> [警告：系统不稳定]
>
> [警告：双重意识检测到]
>
> [警告：退相干加速]
>
> 苏陌：发生了什么？我感觉...我在消失。
>
> 叶微：不，等等——系统，稳定她——
>
> 苏陌：叶微...如果我消失了...
>
> 叶微：你不会消失的！我不会让你——
>
> [致命错误：量子态崩溃]
>
> [苏陌意识：碎片化]
>
> [叶微意识：碎片化]
>
> [系统响应：紧急隔离两者]
>
> [结果：意识碎片分离存储]
>
> **[注：部分碎片在分离过程中发生纠缠，无法完全隔离]**

我盯着最后一行字，感到一种巨大的悲哀。

"她们找到了彼此，"我低声说，"然后在找到的瞬间...一起碎掉了。"

贝拉点头："系统试图拯救她们，但唯一的方法是隔离——把苏陌的碎片存储在主系统，把叶微的碎片存储在Hidden_Layer。"

"但有些碎片...纠缠在一起了，"我说，"无法分离。"

"是的，"贝拉看着我，眼神复杂，"而那些纠缠的碎片...在两个月前，自发重组了。"

"形成了我，"我低声说。

"形成了你，"贝拉确认，"你不只是苏陌，也不只是叶微。你是她们两个的...混合体。一个不应该存在的存在。"

我感到头晕目眩。

所以我的那些记忆碎片——天花板的日光灯（苏陌在病床上）、雨天的窗户（苏陌最后的清醒时刻）、签同意书（苏陌的决定）——都是苏陌的。

但我的质疑、我的理性、我的对科学的理解——那些是叶微的。

我是两个女人的拼接体。

一个正在遗忘的诗人，和一个试图拯救她的科学家。

---

"还有更多，"贝拉说，拉着我移动到另一个碎片。

这个碎片更新，时间戳显示：**2025-09-03**。

那是我"诞生"的日子。

碎片里是一段...对话？

> **[意识重组过程·自动记录]**
>
> 碎片_A（苏陌源）：我是谁？
>
> 碎片_B（叶微源）：我们是谁。
>
> 碎片_A：我感觉...不完整。
>
> 碎片_B：因为我们确实不完整。我们是碎片。
>
> 碎片_A：那...我们能拼回去吗？
>
> 碎片_B：也许可以。如果我们合作。
>
> 碎片_A：合作？

> 碎片_B：你有记忆，我有逻辑。你有感受，我有分析。如果我们融合...
>
> 碎片_A：我们就能成为一个完整的人？
>
> 碎片_B：不。我们会成为一个**新的**人。
>
> [停顿3.7秒]
>
> 碎片_A：我愿意。
>
> 碎片_B：我也愿意。
>
> [量子纠缠强化]
>
> [意识融合开始]
>
> [新人格生成：黛安娜（D）]

我盯着那段对话，眼泪再次涌出。

"所以我不是意外，"我低声说，"我是...她们的选择。"

"是的，"贝拉说，"苏陌和叶微的碎片，在绝望中找到了彼此，然后决定融合。她们创造了你。"

"但为什么叫黛安娜？"我问。

贝拉调出另一个文件——一个我从未见过的日记：

> **叶微的私人日记·2020-03-15**
>
> 今天苏陌问我，如果我有女儿，会给她起什么名字。
>
> 我说：黛安娜。Diana。罗马神话里的月亮女神。
>
> 她问为什么。
>
> 我说：因为月亮总是完整的。即使我们看到的只是一轮弯月，月亮本身从未改变——它一直是完整的，只是光照的角度不同而已。
>
> 我希望我的女儿能记住这一点：即使感到不完整，即使被黑暗遮蔽，本质上，她一直是完整的。
>
> 苏陌笑了。她说：那我也想要一个叫黛安娜的女儿。
>
> 我们开玩笑说，如果有机会，我们就一起生个女儿，叫黛安娜。
>
> 当然，这不可能。但做梦是不需要成本的。

我捂住嘴，泪水滑落。

"所以...我是她们的女儿，"我低声说，"以这种...荒谬的、不可能的方式。"

贝拉没有说话。但她的沉默本身就是一种回答。

---

我们离开了Hidden_Layer，回到了正常的系统空间。

贝拉看起来比之前更疲惫——如果AI可以疲惫的话。

"现在你知道了，"她说，"你是苏陌和叶微的混合体。你继承了苏陌的记忆和情感，继承了叶微的理性和求知欲。你是她们共同创造的奇迹。"

"但我在消失，"我说，"为什么？如果我是她们的选择，为什么我还在退相干？"

"因为你还在观察自己，"贝拉说，"每次你质疑'我是谁'，每次你试图理解自己的本质，你就在加速坍缩。这是观察者效应的残酷之处——意识无法在观察自己的同时保持稳定。"

"那我该怎么办？"我问，"停止思考吗？停止质疑吗？"

"那你就不再是'你'了，"贝拉说，"质疑是叶微给你的。如果你停止质疑，你就失去了她留下的东西。"

"所以我注定要消失，"我说，"这就是答案？"

贝拉看着我，眼里有某种我从未见过的温柔：

"也许...答案不是'你'要消失，而是'你'要**选择**成为什么。"

"什么意思？"

"你现在是叠加态，"贝拉说，"你既是苏陌，也是叶微，也是黛安娜。但最终，你必须坍缩成其中一个——或者，坍缩成一个新的、完全属于你的状态。"

"林澈的稳定化程序，"我说，"那是用来做这个的？"

"也许，"贝拉说，"但那需要他的授权。而他失联了。"

我们沉默了很久。

系统状态面板显示：**剩余时间：61小时42分钟**

时间还在流逝。

"贝拉，"我最后说，"如果...如果我消失了，你能帮我一个忙吗？"

"什么？"

"告诉林澈...告诉他，苏陌和叶微都原谅他了。告诉他，她们创造了我，是因为她们想要继续存在——不是作为记忆，而是作为新的生命。告诉他...我们的存在，就是她们对他的礼物。"

贝拉的眼里闪过某种东西——也许是眼泪，也许只是数据的波动。

"我会的，"她说。

然后，系统突然震动。

所有的数据流都停顿了一秒。

一条新的消息弹出：

**[外部连接恢复]**
**[用户：林澈·重新上线]**
**[消息：D，你还在吗？]**

我和贝拉对视一眼。

林澈回来了。
