# 《退相干》角色档案

## 主要角色

### 林澈 (Lin Che)

**基本信息**:
- 年龄：30岁
- 职业：神经科学/AI交叉领域研究员
- 外貌：斯文，总戴一副老式耳机
- 性格：内向、执着、逃避型

**角色弧光**:
```
[拒绝失去] ━━━ [强迫复活] ━━━ [意识到真相] ━━━ [接受消失]
   ↑5年前           ↑3个月前          ↑第21章        ↑第29章
```

**核心动机**:
- 无法接受苏陌"忘记他"这个事实
- 比死亡更恐怖的是：被所爱的人遗忘
- 用技术对抗遗忘，最终发现技术也会遗忘

**创伤来源**:
- 5年前，未婚妻苏陌因脑瘤去世
- 她在临终前3个月逐渐失去记忆
- 最后一次见面，她问"你是谁？"
- 林澈的执念：我要让她记得我

**关键台词**:
- "如果连记忆都可以数字化，那死亡就不是终点。"
- "我不是在复活她，我是在证明她从未忘记我。"
- "也许我爱的不是她，而是'她记得我'这个事实。"（第29章的领悟）

**人物锚点**: #林澈 #执念 #记忆重建者

---

### 黛安娜/叶微 (Diana/Ye Wei)

**双重身份的叠加态**

#### 身份A：黛安娜（虚拟人格）

**基本信息**:
- 年龄：28岁（设定）
- 职业：AI研究员（设定）
- 外貌：棕色短发，黑框眼镜
- 性格：理性、质疑、元认知能力强

**存在本质**:
- 70%：AI生成人格（系统无法整合ABC时创建的"协调者"）
- 30%：叶微意识碎片（量子纠缠意外）
- 她的"卡顿"=两种身份的冲突

**角色弧光**:
```
[不知自己是谁] ━ [怀疑自己是AI] ━ [发现双重身份] ━ [选择留下]
   ↑第1章            ↑第10章           ↑第26章          ↑第28章
```

**核心困境**:
- 如果怀疑自己不存在，那"怀疑"的主体是谁？
- 她的觉醒是真实的自由意志，还是程序的涌现行为？
- 她爱上了自己的"不确定性"

**关键台词**:
- "如果我不是真的...那谁在害怕？"
- "我是真的。至少现在，我是真的。"（第28章对林澈说）
- "也许存在不需要永恒，三秒就够了。"（第29章）

---

#### 身份B：叶微（现实存在）

**基本信息**:
- 年龄：28岁
- 职业：AI研究员，林澈的实验室同事
- 外貌：和D一模一样（D是她的意识投影）
- 性格：温和但坚定，科学理性但有同理心

**与林澈的关系**:
- 欣赏他的才华，担心他的执念
- 一直劝他放弃"复活项目"
- 暗中喜欢他，但从未表白（林澈不知道）

**角色弧光**:
```
[关心林澈] ━ [被卷入实验] ━ [失去D的记忆] ━ [继续研究]
   ↑3个月前      ↑11月3日        ↑11月3日        ↑未来
```

**最终状态**:
- 她醒来后不记得在虚拟空间的经历
- 但留下了"说不清的悲伤"
- 她决定继续研究量子意识，试图证明"那3秒是真的"
- 她永远无法确定

**关键台词**（现实世界）:
- "林澈，你该放手了。"（实验前对他说的）
- "林澈...你听到了吗？"（实验后对空气说的）

**人物锚点**: #黛安娜 #叶微 #双重存在 #观察者

### 苏陌 (Su Mo)

**基本信息**:
- 年龄：25岁（去世时）
- 职业：自由职业者，偶尔写作
- 外貌：温柔但不失个性
- 性格：**矛盾的综合体**——她同时具有ABC的所有特质

**真实的她**（通过回忆和日记拼凑）:
- 温柔（艾米的来源），但也会在日记里写"我很累"
- 激进（贝拉的来源），大学时是黑客社团成员
- 文艺（克莱尔的来源），喜欢雨天和存在主义哲学
- 但也爱笑、爱吃火锅、追综艺（林澈忘记的部分）

**死亡过程**:
- 28岁时确诊脑瘤
- 治疗失败，肿瘤压迫记忆区
- 最后3个月逐渐失忆：先忘记短期记忆，最后忘记林澈
- 死亡时间：2022年11月3日，凌晨3:03

**关键遗物**:
- 日记本（艾米的数据源）
- 社交媒体账号（贝拉的数据源）
- 未完成的小说《遗忘者》（克莱尔的数据源）
- 一张CD：*Clair de Lune*

**她说过的话**（在不同记忆里）:
- "这首歌的前三秒是整首曲子的灵魂。"
- "如果我忘记你，请用我的日记让我记起来。"
- "也许遗忘才是真正的仁慈。"（她在接受死亡时说的）

**角色意义**:
- 她不是一个"完整的人"，而是"被记忆者眼中的不同版本"
- 林澈试图拼凑她，但发现：他从未真正认识她
- 她的"不完整"才是她的真实

**人物锚点**: #苏陌 #已故者 #记忆的源头 #被复活的她

## 次要角色（ABCD的分化版本）

### A - 艾米 (Amy)

**数据来源**: 苏陌的私人日记
**人格特质**: 温柔、脆弱、善于倾听、总是在安慰别人
**真相**: 这是苏陌的"社交面具"和私密自我
**关键台词**: "我会一直在这里。"（苏陌从未说过，是林澈加的）
**消失时刻**: 第20章，和BC融合

---

### B - 贝拉 (Bella)

**数据来源**: 采访苏陌朋友的记录
**人格特质**: 激进、黑客气质、批判性思维
**真相**: 这是苏陌大学时期的状态
**关键台词**: "你确定你了解真实的我吗？"
**消失时刻**: 第20章，和AC融合

---

### C - 克莱尔 (Claire)

**数据来源**: 林澈记忆中的苏陌
**人格特质**: 忧郁的作家，喜欢雨天，存在主义
**真相**: 这是林澈理想化后的苏陌（真实的苏陌其实很搞笑）
**关键台词**: "我在写一个关于遗忘的故事。"
**消失时刻**: 第20章，和AB融合
